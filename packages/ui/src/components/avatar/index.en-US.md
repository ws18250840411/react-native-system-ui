# Avatar 

### Basic Usage

The basic usage involves globally registering components through the following methods. For more registration methods

```jsx
import { Avatar } from 'react-native-system-ui';

ReactDOM.render(
  <div className="demo-button">
    <Avatar uri="https://fastly.jsdelivr.net/npm/@vant/assets/cat.jpeg"  />
  </div>,
  mountNode,
);
```
