# TextInputBase（规划中）

> 该组件尚未在 `react-native-system-ui` 中正式发布，目前仅作为内部技术探索。请使用 `TextInput` 或 `Field.TextInput` 获取同等能力。

## 状态说明

- 代码仓库中保留了 TextInput 底层实现草稿，未来可能提供更细粒度的基础输入组件，用于完全自定义的输入框拼装。
- 当前 npm 包并未导出该组件，也没有可用的 API。
- 当计划确定后，会更新文档并发布版本公告。

## 可替代方案

- 普通输入：`TextInput`
- 表单友好版本：`Field.TextInput` / `Field.NumberInput`
- 数字场景：`NumberInput`

如需提前参与设计或提供需求，请在项目 issue 中反馈。
