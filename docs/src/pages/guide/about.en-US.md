# Introduction

RN System UI is a design system and component library crafted for React Native. It focuses on delivering consistent cross-platform experiences with a unified design language, theming tokens, and engineering guidance.

## Goals

- Keep product experiences aligned across mobile and web touchpoints
- Give teams a standardized set of components and interaction patterns
- Reduce collaboration friction between design and engineering

## Capabilities

- Comprehensive base and composite components covering common scenarios
- Extensible theme and token system for brand customization
- Design assets and best-practice playbooks shipped alongside components
