# RN System UI

React Native 生态的设计系统与组件库，提供一致的视觉语言、主题体系与工程实践。

## 核心特性

- 移动优先的交互体验，覆盖 App 与 Web 场景
- 50+ 高质量组件与 700+ 主题变量
- TypeScript 原生支持，完善的类型提示与调试体验
- 深浅色主题切换与品牌化能力

## 快速上手

1. 安装依赖：`npm install react-native-system-ui`
2. 在应用入口包裹 `Provider` 并初始化主题
3. 引入所需组件与样式，开始构建体验

## 生态能力

- 提供 Figma / Sketch 设计资产，与工程组件保持同步
- 支持 Tailwind 风格的主题定制方案
- 提供示例与模板，加速产品落地
