# RN System UI

A design system and component library for the React Native ecosystem, delivering a unified visual language, theming model, and engineering workflow.

## Highlights

- Mobile-first interaction patterns across app and web surfaces
- 50+ polished components with 700+ theme tokens ready to customize
- First-class TypeScript support with rich editor hints
- Built-in dark mode and brandable theming primitives

## Quick Start

1. Install dependencies: `npm install react-native-system-ui`
2. Wrap your app with the `Provider` and configure the theme
3. Import the components you need and start composing experiences

## Ecosystem

- Figma and Sketch design assets aligned with the component library
- Tailwind-style theme customization for rapid iteration
- Starter examples and templates to accelerate delivery
